CSS INTRODUKTION

Vad är CSS?


  • CSS står för Cascading Style Sheets.
  • CSS beskriver hur HTML element ska visas.
  • Med hjälp av CSS så sparar du massa tid. En CSS fil kan kontrollera flera hemsidor samtidigt.

    • Hur kan ett simpelt CSS dokument se ut?

    Exempel på hur det kan se ut:

    • body {
    • background-color: lightblue;//Ger dokumentets kropp bakgrundsfärgen ljusblå
    • }

    • h1 {
    • color: orange;//Ger h-taggen färgen orange på texten
    • text-align: center;//Placerar h-taggen i centrum på hemsidan
    • }

    • p {
    • font-family: verdana;//Ger p-taggen textstilen "verdana"
    • font-size: 16px;//Ger p-taggen en textstorlek på 16px
    • }

HTML INTRODUKTION

Hur kan ett simpelt HTML dokument se ut?

Exempel på hur det kan se ut:

  • <!DOCTYPE html >
  • <html>
  • <head>
  • <title>Rubrik</title>
  • </head>
  • <body>

  • <h1>Min första rubrik</h1>
  • <p>Min första paragraf</p>

  • </body>
  • </html>

Vad händer i exemplet?

  • < !DOCTYPE html> deklarerar att dokumentet är ett HTML5 dokument
  • < html> elementet är rotelementet på en HTML-sida
  • < head> elementet innehåller metainformation om HTML-sidan
  • < title>anger en titel för HTML-sidan och visar namnet på sidans flik
  • < body>definierar dokumentets kropp och är en behållare för allt synligt innehåll, såsom rubriker, bilder och stycken med mera.
  • < h1>definierar en stor rubrik som visas på sidan
  • < p>definierar ett stycke som syns på sidan
